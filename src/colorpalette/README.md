# colorpalette

```txt
$ use colorpalette -h
ryota2357's color palette (using material color).

Options:

  --help   -h
      Show help.

  --name   -n
      Display color code.

  --text   -t  [args: String]
      Use specified text as text of forground color sample.
      If use with --name options, this option is ignored.

  --bg     -b  [args: String]
      Set background color of normal text.
      Defaut: "#ffffff"
```
